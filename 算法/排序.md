## 排序算法

- 冒泡排序 两两比较(往后移)
```js
var arr = [0,3,4,5,1,9];

function bubbleSort(arr){
    var len = arr.length;
    for(var outer = len; outer >= 2;outer--){
        for(var inner = 0;inner < outer;inner++){
            if(arr[inner] > arr[inner+1]){
                [arr[inner],arr[inner+1]] = [arr[inner+1],arr[inner]]
            }
        }
    }
    return arr;
}
const result = bubbleSort(arr);
console.log(result)

```

- 选择排序 遍历自身以后的元素，最小的元素跟自己调换位置(往前移)

```js
function selectSort(arr){
    var len = arr.length;
    for(var outer = 0;outer < len;outer++){
        for(var inner = outer;inner < len;inner++){
            if(arr[outer] > arr[inner]){
                [arr[outer],arr[inner]] = [arr[inner],arr[outer]]
            }
        }
    }
    return arr;
}
result = selectSort(arr);
console.log(result);
```

- 插入排序：即将元素插入到已排序好的数组中

```js
function insertSort(arr) {
    for(let i = 1; i < arr.length; i++) {  //外循环从1开始，默认arr[0]是有序段
        for(let j = i; j > 0; j--) {  //j = i,将arr[j]依次插入有序段中
            if(arr[j] < arr[j-1]) {
                [arr[j],arr[j-1]] = [arr[j-1],arr[j]];
            } else {
                break;
            }
        }
    }
    return arr;
}
```

- 乱序 - Fisher–Yates
```js
function shuffle(arr){
    let i,len = arr.length;
    for(i =len;i;i--){
        let j = Math.floor(Math.random()*i);
        [arr[j],arr[i]] = [arr[i],arr[j]]
    }
    return arr;
}
```
